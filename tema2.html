Tema 2 - Grundlæggende web

I dette tema lærte vi om HTML og CSS. Vi fik helt grundlæggende viden omkring hvordan man sætter sin HTML op og hvordan man kan style det med CSS. Der var især fokus på box modellen som er en essentiel ting når man skal udvikle sit site. Hertil lærte vi bl.a. om margin og padding.

Vi arbejdede mobile-first hvor vi skulle sætte indhold ind i vores HTML ud fra wireframes tildelt af underviserne. Herefter lærte vi om grid og media queries, hvor vi ud fra layoutdiagrammer skulle tilpasse det til desktop versionen.

Designprincipper, gestalt og stilanalyse

Vi lærte om forskellige designprincipper såsom gestaltlove og hvordan man analysere en stil. Vi fik til opgave at læse forskellige kapitler i bogen
”Interfacedesign – Fra idé til prototype” af Morten Rold. Den gav mig et godt indblik i hvordan man opfatter webdesign og generelt opsætning af websites vha. grid (kolonner og rækker) lige såvel visuelt hierarki og kontraster.
